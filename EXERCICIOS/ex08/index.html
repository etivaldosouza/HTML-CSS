Texto deletado

podemos marcar um texto como excluido utilizando a tag < del > para indicar que ele deve ser lido,mas não considerado.

Texto inserido

podemos marcar um texto como inserido utilizando a tag < ins > para dar uma enfase e indicar que ele foi adicionado depois.

existe também o sublinhado com a tag < u > (não semântico).

Texto Sobrescrito

para inserir coisas do tipo: x20+3 utilizando a tag < sup >.

Texto Subescrito

para inserir coisas do tipo h20 utilizando a tag < sub >.
